Soccer Community Hub – Site Plan

Soccer DAY

Soccer Community Hub

This name represents a central hub for local amateur soccer players. It communicates that users can find training information, event details, and community activities all in one place.

Optional domain: soccerhub-community.org

Site Purpose

The purpose of this site is to provide helpful information and resources for people who enjoy playing soccer. The site will include local game schedules, basic training tips, beginner-friendly rule explanations, and ways to join local amateur teams or pick-up matches. The goal is to make it easier for users to engage with the soccer community.

Scenarios

  • “Are there any local pick-up soccer games that beginners can join?”
  • “How can I practice basic soccer skills like passing, dribbling, and shooting?”
  • “Where can I find soccer fields near my area?”

Color Scheme

Primary Color: #1a73e8 (Blue) – used for main headings, buttons, and accent elements.

Secondary Color: #0d1b2a (Dark Navy) – used for body text and subheadings.

Background Color: #f5f7fa (Light Gray) – used for the page background.

These colors are applied throughout this document.

Typography

Roboto – used for body text.

Roboto Slab – used for headings (h1, h2).

A simple two-font system was chosen for clean readability.
